# SpringBoot大学生租房平台 python991

## 项目概述

本项目是基于SpringBoot的大学生租房平台，旨在为大学生提供便捷的租房服务。用户可以通过系统进行房源查看、注册登录、订单管理等功能。管理员可以对系统进行全面的管理。

## 技术栈

### 开发环境

- IDEA
- JDK 1.8
- MySQL
- Maven
- Navicat

### 技术框架

- 架构：B/S、MVC、Web框架
- 技术栈：Java、SpringBoot、Mybatis、Html、Css、Vue、MySQL

## 功能模块

### 用户功能

- 注册登录
- 首页浏览
- 房源信息查看
- 个人中心
- 订单信息管理
- 订单配送管理

### 管理员功能

- 系统全面管理
- 个人中心
- 会员管理
- 房源分类管理
- 房源信息管理
- 订单信息管理
- 订单配送管理
- 管理员管理
- 系统管理

## 系统角色

- 普通用户
- 管理员

## 运行环境

- 服务器：推荐使用Linux系统
- 数据库：MySQL
- 前端：支持现代浏览器

## 目录结构

```
SpringBoot大学生租房平台
├── backend              # 后端代码
├── frontend             # 前端代码
└── sql                  # 数据库文件
```

## 部署步骤

1. 准备好开发环境（IDEA、JDK 1.8、MySQL等）
2. 导入数据库文件至MySQL数据库
3. 使用IDEA打开后端代码，配置好Maven依赖
4. 编译并运行后端项目
5. 使用WebStorm或其他工具打开前端代码
6. 前端项目构建完成后，部署至服务器
